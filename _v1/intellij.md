# Basic hotkeys

```text
Ctrl + Shift + A          # Search for commands/hotkeys

Ctrl + B                  # Go to declaration - can also Ctrl + click
Ctrl + Q                  # See quick docs
Ctrl + Shift + Up/Down    # Move highlighted lines up/down
Ctrl + P                  # Brings up valid params

Ctrl + Shift + Backspace  # Last edited location
Ctrl + Alt + Left/Right   # Last positions
Ctrl + Shift + F7         # Find/highlight usages of var in file - F3/Shift + F3
Alt + F7                  # Find all usages of var, method etc.

Ctrl + Shift + V          # Choose recent items in clipboard
Alt + Up/Down             # Move between methods
Shift + F6                # Refactor/rename
Ctrl + Y                  # Delete current line
Ctrl + Shift + F          # Search everywhere
Ctrl + Shift + Z          # Redo
Ctrl + Shift + U          # Upper/lower case toggle

Ctrl + Shift + F12        # Show editor page only
Alt + Left/Right          # Move left/right between tabs
```
